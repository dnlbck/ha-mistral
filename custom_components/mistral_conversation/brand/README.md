# Brand Assets

HACS requires brand images for custom integrations.

## Required
- `icon.png` — 256x256 square icon (used in integration lists and small UI elements)

## Optional
- `logo.png` — Rectangular logo (used on the integration page)
- `icon@2x.png` — High-DPI icon (512x512)
- `logo@2x.png` — High-DPI logo

## How to add
1. Create or obtain a square icon representing Mistral AI (256x256 PNG)
2. Place it as `icon.png` in this directory
3. Optionally add a rectangular `logo.png`

## Notes
- Use a transparent background
- Keep the design simple and recognizable at small sizes
- If using the Mistral AI logo, verify you have permission per their brand guidelines
